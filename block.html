Введение

block - это ключевое слово для создания блоков(функций).

Синтаксис

Синтаксис block в Dark выглядит следующим образом:

block имя_блока =>
> обычные_команды
endblock

Примеры использования

Простой блок:

Dark: 

block test_block =>
> output Простой блок
endblock
run test_block
