Введение

if - это ключевое слово для создания проверок условий переменной.

Синтаксис

Синтаксис if в Dark выглядит следующим образом:

if данные_или_[{переменная}] => обычная_программа_с_разделением_строк_через_\|/

Примеры использования

Сравнение переменных:

Dark: 

<-#-> Объявляем переменную "dighit":
set int dighit 123
<-#-> Примеры сравнения чисел:
if [{dighit}] == 123 => output True

if [{dighit}] < 123 => output False

if [{dighit}] > 123 => output True

Простое условие:

Dark: 

<-#-> Объявляем переменную "dighit":
set int dighit 123
<-#-> Пример простого условия:
if [{dighit}] == 123 => output True

Сложное условие:

Dark: 

<-#-> Объявляем переменную "dighit":
set int dighit 123
<-#-> Пример сложного условия:
if [{dighit}] == 123 => set input str text \|/ output text = [{text}]

Булевые проверки:

Dark: 

<-#-> Объявляем булевую переменную "test":
set bool test True
<-#-> Пример булевой проверки:
if [{test}] => output [{test}]
